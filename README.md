## サイト概要
家計や料理のレシピなど生活に必要な情報を整理しやすく、
備忘録としても保存しておけるようなサイトです。
マイレシピを文章などで分かりやすく保存して
誰かに伝えやすくすることもできます。
定期的にかかる費用を登録することで
月当たり、年あたりを計算することもできます。

### サイトテーマ
家計の補助をし、料理のレシピを共有しあうSNSサイト

### テーマを選んだ理由
母親がいつも家事を頑張っていながらも仕事もしているので、
楽にできるような便利な物を作ってあげたいと思い、
生活を少しでも豊かにするためです。

### ターゲットユーザ
*主婦・主夫の方
*一人暮らしの方

### 主な利用シーン
*家計の管理
*レシピを保存、閲覧、公開など
